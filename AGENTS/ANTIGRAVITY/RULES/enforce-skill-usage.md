# Enforce Antigravity Skill Usage

To ensure consistency and safety in this project, you MUST use the following skills for their respective tasks:

## Git Commits
When asked to commit changes or write a commit message, you MUST use the `git-commit-formatter` skill to format the message according to the Conventional Commits specification.

## Database Schema Changes
When asked to validate or modify SQL schema files (e.g., `.sql` files), you MUST use the `database-schema-validator` skill. Do not rely on manual inspection for schema validation.

## Scaffolding ADK Tools
When asked to create a new custom Tool for the Agent Development Kit (ADK), you MUST use the `adk-tool-scaffold` skill to ensure the correct boilerplate and structure.
